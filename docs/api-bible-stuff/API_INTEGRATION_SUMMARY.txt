ğŸ“ HOW IT WORKS:
================================================================================

1. App requests verses for a chapter
2. Hook checks if API is enabled (useApiIfAvailable: true)
3. If API is enabled:
   - Tries to fetch from REST API
   - If successful, returns verses with source: 'api'
   - If fails, falls back to local JSON
4. If API is disabled:
   - Uses local JSON files (current behavior)
5. Verses are automatically parsed into your format:
   { '1': 'verse text', '2': 'verse text', ... }
